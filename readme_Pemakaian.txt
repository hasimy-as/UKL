Di "readme" ini, ditunjukkan cara untuk menggunakan projek ini.
Berikut langkah-langkahnya:

-----------------------------------------------------------------------------------------------------

1. Jalankan projek dengan kode berikut:
    `node serverMain.js` atau `nodemon serverMain.js`

    Jika ingin menjalankan dengan simple, ketikkan kode berikut ke Terminal:
    `npm start`

2. Buka web browser anda, dan silahkan akses `localhost:5000/`.

3. Setelah terakses, akan ada 2 prompt penggunaan, yaitu User atau Admin.
    Jika pengguna pilih user, maka harap login dengan kredensial sebagai berikut:
	Email: hasimy@telkom-sarpra.sch.id
	Password: user1
    Dari sini, pengguna dapat memesan barang yang tersedia di Inventaris Sarpra SMK Telkom Malang/


    Jika pengguna pilih user, maka harap login dengan kredensial sebagai berikut:
	Email: hasimy_admin@adminsarpra.com
	Password: admin1
    Dari sini, pengguna dapat melihat para peminjam barang Sarpra, dan dapat menerima atau menolak permintaan peminjaman peminjam.


-----------------------------------------------------------------------------------------------------
